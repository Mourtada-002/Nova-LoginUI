# Nova Login UI

A **modern login interface built with Flutter**, combining an immersive purple background with elegant glassmorphism effects for a sleek and refined authentication experience.

## Features

**Immersive Purple Background** – Deep violet tones creating a bold and atmospheric visual identity  
**Glassmorphism Input Fields** – Frosted glass styling on text fields for a modern, layered aesthetic  
**Polished Auth Button** – Elegant call-to-action designed to complement the overall UI  
**Responsive & Customizable** – Adaptive layout easily tailored to any project  

## Technologies Used

- **Flutter** – Cross-platform UI framework
- **Dart** – Application logic and widget structure
